a) Partie : Mon premier exemple (Page 2/10)

La balise h1 permet de mettre le titre du 1er paragraphe, la balise h2 permet de mettre le titre du 2ème paragraphe et la balise p permet de mettre le contenu des paragraphes.

La différence entre les formats GIF, JPEG et PNG :

  • Le format GIF est un format d’image numérique couramment utilisée sur le WEB ;
  • Le format JPEG est un format d’image numérique, qui à été crée pour remplacer le format GIF (avant propriétaire) et dont la compression était soumise à un brevet ;
  • Le format JPEG est une norme qui définit le format d’enregistrement et l’algorithme de décodage pour une représentation numérique compressée d’une image fixe.
